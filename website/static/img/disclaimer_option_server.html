Conditions d’utilisation du module optionnel « Serveur sécurisé »

Description

Le module optionnel « Serveur sécurisé » permet une sauvegarde automatique de toutes les données de votre application Diabphone sur un serveur sécurisé. Vos données sont cryptées, stockées sur nos serveurs et associées à un code personnel, anonyme, unique et choisi aléatoirement. Ce module permet également une synchronisation automatique de vos différents appareils (iPad, iPhone, etc.) sur lesquels l’application Diabphone est installée. Il est possible de visualiser votre carnet de surveillance, et de l’imprimer, depuis une simple page web, via le site www.diabphone.com, grâce à votre code personnel et à partir d’un code d’accès, que vous seul pouvez générer et qui déterminera la durée maximale durant laquelle votre carnet est accessible par internet. En cas de perte ou de vol de votre appareil, une procédure de restauration vous permet de retrouver toutes les données et options votre application Diabphone. Une fois achetée, l’option « Serveur sécurisé » a une validité d’une année.

Avertissements et recommandations

Utilisation du module optionnel « Serveur sécurisé » et accès à Internet

Le module optionnel « Serveur sécurisé » requiert que vos appareils disposent d’un accès à Internet, par Wi-Fi ou 3G, pour la sauvegarde et la synchronisation de vos données ou l’affichage de votre carnet depuis le site www.diabphone.com. De ce fait, l’utilisation de ces fonctions est impossible lorsque vous vous trouvez dans une situation ou une zone géographique ne vous offrant pas d’accès à Internet (absence de Wi-Fi ou de réseau 3G accessible), ou si les caractéristiques de vos appareils ne vous permettent pas d’accéder à Internet.

Diabphone© décline toute responsabilité en cas de panne, indisponibilité, ou quelconque problème survenant sur les serveurs distants utilisés par le module optionnel « Serveur sécurisé », et ne serait être tenu pour responsable d’aucun dommage d’aucune sorte dû à l’utilisation ou à l’impossibilité d’utiliser ce module. L’utilisateur s’engage à ne se prévaloir d’aucun préjudice dans un tel cas de figure.

Code personnel et anonymat des données

Vos données stockées sur le serveur sécurisé sont identifiées à partir de votre code personnel. Ce code anonyme permet l’anonymat de vos données. Conservez le précieusement et ne le divulguez pas sans raison. La divulgation de ce code permet aux personnes qui en ont connaissance d’établir un lien entre vous-même et des données identifiées par votre code personnel, auxquelles elles pourraient avoir accès : de ce fait, l’anonymat de ces données disparait.

Diabphone s’engage expressément à ne jamais dévoiler à quiconque les données personnelles des utilisateurs qui peuvent être portées à sa connaissance.

Accès au carnet de surveillance via www.diabphone.com

Le module optionnel « Serveur sécurisé » permet de visualiser votre carnet de surveillance via le site www.diabphone.com, à la rubrique « Accès médecin ». Cette fonctionnalité a été principalement conçue pour permettre à votre diabétologue ou à votre médecin traitant de visualiser vos résultats, à distance si besoin, si vous le souhaitez. Il faut pour cela disposer de votre code personnel et d’un code d’accès, que vous seul pouvez générer. Ce code d’accès change chaque jour automatiquement et varie en fonction de la durée durant laquelle vous souhaitez que votre carnet soit accessible par ce moyen (de 1 jour à 1 semaine au maximum). Si vous confiez votre code personnel et un code d’accès à autrui, l’accès à l’ensemble des données de votre carnet de surveillance, y compris les commentaires y figurant, lui sera possible jusqu’à l’expiration du délai que vous aurez fixé.

Ne confiez pas votre code personnel et un code d’accès sans raison ou à une personne qui n’est pas l’un de vos professionnels de santé. Vous gardez toujours la possibilité de ne pas utiliser cette fonctionnalité du module optionnel « Serveur sécurisé » et de transmettre les résultats de votre carnet de surveillance par un autre moyen. Diabphone ne saurait en aucun cas être tenu pour responsable des conséquences liées à la diffusion, de votre part, de votre code personnel, seul ou associé à un code d’accès. La diffusion à autrui de votre code personnel, seul ou associé à un code d’accès, relève de votre propre décision. Utilisez votre bon sens et votre jugement avant de prendre cette décision.

Le module optionnel « Serveur sécurisé » fait partie intégrante du logiciel Diabphone. À ce titre vous reconnaissez avoir lu et accepté intégralement et sans réserve le Contrat général d’utilisation de l’application Diabphone©. Dans le cadre du module optionnel « Serveur sécurisé », », l’accès aux données du carnet de surveillance peut se faire via le site www.diabphone.com. Vous acceptez donc expressément et sans réserve par la présente les conditions d’utilisation du site web www.diabphone.com.
